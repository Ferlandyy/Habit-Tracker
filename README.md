# Habit Tracker

Add habits, tick your day, and keep your streaks going. Built with **Vanilla JS (no frameworks)**. Data persists in **localStorage**.

## Live Demo
- **Pages**: https://YOUR_USERNAME.github.io/habit-tracker/  
- **Repository**: https://github.com/YOUR_USERNAME/habit-tracker

> Replace `YOUR_USERNAME` after deployment. If you prefer Render, put that link here too.

---

## Features
- Add a habit
- Toggle last **7 days** (D-6 … Today)
- **“Tick today”** shortcut
- Live **current streak** calculation
- Delete with confirm
- Responsive, keyboard-accessible UI (visible focus, labels, contrast)

---

## Tech Stack
- HTML, CSS, **Vanilla JavaScript**
- **localStorage** (JSON stringify/parse)
- Optional: GitHub Pages for hosting
